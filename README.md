# Pet Management System

Ένα απλό πρόγραμμα γραμμής εντολών (CLI) σε Python για τη διαχείριση ζώων προς υιοθεσία σε ένα καταφύγιο.
## Δοκίμασέ το άμεσα

Τρέξε το project κατευθείαν στον browser χρησιμοποιώντας το GitHub Codespaces και την εντολή "python main.py" στο terminal:

[![Run in Codespaces](https://github.com/codespaces/badge.svg)](https://github.com/codespaces/new?repo=riasanto97-py/pet-management-system)

Το πρόγραμμα επιτρέπει στον χρήστη να:

* δει όλα τα ζώα
* προσθέσει νέα ζώα
* αφαιρέσει υπάρχοντα ζώα
* αποθηκεύσει και να φορτώσει δεδομένα μέσω JSON

Το project παρουσιάζει:

* modular σχεδίαση
* χρήση αντικειμενοστραφούς προγραμματισμού (OOP)
* αποθήκευση δεδομένων σε JSON
* καθαρό και επαναχρησιμοποιήσιμο κώδικα

# Σημειώσεις για τον Κώδικα

Το αρχείο `main.py` περιέχει το entry point του προγράμματος:

```python
if __name__ == "__main__":
    main()
```

Αυτό διασφαλίζει ότι το πρόγραμμα εκτελείται μόνο όταν το αρχείο τρέχει απευθείας και όχι όταν εισάγεται ως module σε άλλο αρχείο.

Με αυτόν τον τρόπο ο κώδικας παραμένει:

* modular
* επαναχρησιμοποιήσιμος
* σωστά δομημένος

Τα αρχεία `models.py` και `storage.py` λειτουργούν ως modules και μπορούν να χρησιμοποιηθούν και σε άλλα projects χωρίς να εκτελείται αυτόματα το πρόγραμμα.
